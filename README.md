# Telegram Reminder Bot

Бот напоминает, что в профиле заканчиваются видео для отложенных публикаций.

## Запуск

1. Создайте `.env` по примеру `.env.example`.
2. Укажите `BOT_TOKEN`.
3. Установите зависимости:

```powershell
pip install -r requirements.txt
```

4. Запустите бота:

```powershell
python bot.py
```

## Доступ

Ботом может управлять только пользователь из `.env`:

```env
ADMIN_USER_ID=408141472
```

## Использование

- `/start` или `/menu` - открыть меню.
- `Создать профиль` - бот спросит название, chat_id и количество видео в день.
- `Профили` - открыть список профилей и редактирование.
- `Проверить сейчас` или `/check_now` - вручную запустить проверку и списание.

При создании профиля `total_videos_available` по умолчанию равен `0`. После создания его можно изменить через меню профиля.

## Хранение

Профили хранятся в `data/profiles.txt` в формате:

```text
profile_name:group_chat_id1,group_chat_idN:total_videos_available:videos_per_day
```

Каждая строка является отдельным профилем. Напоминания по профилю отправляются только админу и чатам из этой же строки.

## Ограничения

- `profile_name` - строка длиной 1-100 символов, без `:`.
- `group_chat_id` - строка вида `-1005289665576`.
- `total_videos_available` - число 0-100.
- `videos_per_day` - число 0-15.
- Количество чатов на профиль не ограничено.

## Напоминания

Каждый день в 10:00 по Москве бот проверяет профили. Если после сегодняшних публикаций видео не хватит на завтра, бот отправляет уведомление админу в личные сообщения и в связанные чаты, где бот присутствует. В уведомлении показывается, сколько видео будет доступно на следующий день. Ошибки отправки логируются через ErrorHandler и не останавливают проверку остальных профилей.
