# О проекте

Телеграм бот, который должен напоминать о том, что заканчиваются видео которые загрузили в отложенные посты в социальные сети.

## Как работает

Есть всего два этапа: 0. Создание профиля

1. Выбор профиля, добавление количества видео и выставление сколько видео поститься в день в конкретном профиле.
2. Напоминание о том, что на конкретном профиле заканчиваются видео **за день**.

## Пример работы

Я создал профиль `test1`, в нём я обозначил, что загружено 2 видео и в профиль выкладвается 2 видео в день. Видео начинают поститься на следующий день.
На следующий день, в 10 утра, бот напоминает мне что на ЗАВТРА видео уже не хватает, потому что оба доступных видео будут загружены сегодня.
После напоминания, бот списывает из total_videos количество videos_per_day .

Управлять ботом может только пользователь с id 408141472 - помести это в .env и давай пользоваться ботом только ему.

# Стек

## Хранение данных

Чтобы хранить данные используй файлы, которые будет создавать бот (не храни все в оперативной памяти). config файл будет содержать строки подобного вида:
`profile_name:group_chat_id1,group_chat_idN:total_videos_available:videos_per_day` .

- profile_name - имя профиля, задается вручную
- group_chat_id<n> - id чата/чатов, перечисленные через запятую, в которые бот будет публиковать напоминание с конкретного аккаунта.
- total_videos_available - сколько всего видео сейчас доступно.
- videos_per_day - сколько на профиле публикуется видео в день.

# Ограничения

profile_name - строка длиной 1-100 символов.
group_chat_id - строка вида `-1005289665576`.
total_videos_available - число в диапозоне 0-100.
videos_per_day - число в диапозоне 0-15.

Количество чатов на один профиль неограничено.

# Основное правило

Профиль действует **только в рамках своей строки**. Нельзя уведомлять о количестве видео профиля на первой строке в чат из 10-ой строки.

# Значения по умолчанию

prodile_name задается при создании - бот это спрашивает.
group_chat_id - бот спрашивает id чата к которому привязывается профиль.
total_videos_available - по умолчанию 0
videos_per_day - пользователя бот спрашивает при создания профиля, только после этого создается строка в конфиге.

# Редактирование профилей

После того, как профиль создан, он занесен в config файл локально где крутиться бот. Далее, пользователь может открыть меню в котором можно провалиться сначала в профиль, а далее в редактирование его названия, связанного/связанных чатов и количества видео в день.

# Напоминания

После того как настройки выставлены, бот отправляет ВСЕ уведомления админу в личные сообщения и в связанные чаты если он в них присутствует (может не присутствовать, добавь ErrorHandler).
